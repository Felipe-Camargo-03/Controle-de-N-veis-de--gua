# 💧 Sistema de Monitoramento de Reservatório

Este projeto em Python foi criado para praticar o ciclo **Entrada de informação → Processamento →
Saída**.

O programa atraves do monitoramento do nivel de água recebe o nivel atual de água e exibe o nivel com uma variação de cor de acordo com o nivel de gravidade referente e a informação, as mensagens retornam nas cores com as seguintes mensagens:
